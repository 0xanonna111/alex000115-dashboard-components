# UI Component Library (Master Dashboard Expansion)

This repository contains the visual building blocks for the **Alex000115** ecosystem. These aren't just buttons; they are data-reactive components designed specifically for the 100 protocols generated in Phase 1. Each component is self-contained and "Flat," meaning it includes its own logic, styles, and types in a single file or directory for easy portability.

## Core Widgets
* **EscrowProgress:** A visual timeline for milestone-based payments (Repo 81).
* **DarkPoolOrderBook:** An encrypted-style UI for private order matching (Repo 82).
* **IdentityBadge:** A ZK-proof status indicator for decentralized credentials (Repo 94).
* **MeshNodeChart:** A real-time graph of bandwidth pricing for mesh routers (Repo 85).

## Tech Stack
* **Tailwind CSS:** For professional, utility-first styling.
* **Framer Motion:** For fluid, high-end Web3 transitions.
* **Lucide React:** For a clean, technical icon set.

## Setup
1. `npm install`
2. Import components directly into your Next.js `App.js`.
